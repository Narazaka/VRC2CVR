# VRC2CVR

Avatar convertor : VRChat(VRCSDK2) avatar to ChilloutVR avatar

![VRC2CVR](VRC2CVR.png)

## Usage

1. Copy your vrchat avatar project
2. Open the copied project by Unity 2019
3. Import CCK(ChilloutVR SDK)
4. Import VRC2CVR
5. Open avatar scene and do below!
